Run Java-Maven App mit Tomcat via Terminal

Habe hierfür Eclipse EE in Version 2019-03 (4.11.0), tomcat 8.5.42 und Maven 3.6.1 genutzt.

1. Maven App erstellen und als Archetyp WebApp wählen

2. POM.xml erweitern
	2.1 <scope>provided</scope> ergänzen

<dependency>
	<groupId>javax.servlet</groupId>
	<artifactId>javax.servlet-api</artifactId>
	<version>3.1.0</version>
	<scope>provided</scope>
</dependency>

	2.2 In Build folgendes einfügen, App läuft dann unter localhost:8080/myapp

<plugins>
        <plugin>
		    <groupId>org.apache.tomcat.maven</groupId>
		    <artifactId>tomcat7-maven-plugin</artifactId>
		    <version>2.2</version>
		    <configuration>
		        <url>http://localhost:8080/manager/text</url>
		        <server>TomcatServer</server>
		        <path>/myapp</path>
		    </configuration>
		</plugin>
    </plugins>

3. settings.xml von Maven bearbeiten bzw. neu erstellen, liegt in (INSTALL von MAVEN)/.m2/repository

	3.1 Folgendes unter servers einfügen

<server>
        <id>TomcatServer</id>
        <username>admin</username>
        <password>password</password>
</server>

4. Terminal öffnen
5. In Verzeichnis wechseln, wo Projekt liegt
6. mvn clean install laufen lassen
7. mvn tomcat7:run laufen lassen
8. Browser öffnen 
9. localhost:8080/myapp eingeben
10. Hello World Ausgabe ist da
